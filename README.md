# Versão
 **v1.0**
 * 807 Espécies Pokémon com todas as formas alternativas, incluindo Alolan e Megas.

 * Animação Mega Evolução Especial para Clientes Desktop. (Pode ser desativado)

 * Habilidades.

 * Movimentos

 * Cadeia de Evolução.

 * Treinamento, Criação, Digitação e Formulários.

 * Saborear o texto de acordo com os jogos selecionados.

 * Funciona Offline.* :heavy_check_mark:

 * Pode ser instalado :heavy_check_mark:
